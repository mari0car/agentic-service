# API Routes

## Products
- `GET /api/products` -> products/list.md
- `POST /api/products` -> products/create.md
- `GET /api/products/:id` -> products/get.md
- `PUT /api/products/:id` -> products/update.md
- `DELETE /api/products/:id` -> products/delete.md

## Sales
- `GET /api/sales` -> sales/list.md
- `POST /api/sales` -> sales/create.md
- `GET /api/sales/:id` -> sales/get.md
